# academind-nodejs-asgmt-4

User list app, using EJS
#### [Live demo](https://user-list-asgmt-4.onrender.com)

## Instructions
1. Install dependencies - `npm install`
2. Run in development - `npm run dev`
3. Run the app - `npm start`
